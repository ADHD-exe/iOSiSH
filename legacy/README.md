# Legacy reference files

This directory holds repo-root shell files that were kept only as historical reference during the refactor away from repo-managed dotfiles.

They are **not** part of the active install path. Do not copy them directly into a user's home directory.

Do not copy these files into a home directory as-is. They are preserved only to document the old repo-root layout and should not be part of active installer, test, or release paths.
